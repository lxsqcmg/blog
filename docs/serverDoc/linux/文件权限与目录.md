# 文件权限与目录

## 文件属性

```sh
# 切换到 root 账户
su -

# 查看当前目录下的文件数据
ls -al
```

| 命令     | 备注                                                                                  |
| -------- | ------------------------------------------------------------------------------------- |
| `su -`   | 切换到 root 身份                                                                      |
| `exit`   | 回到你自己的身份                                                                      |
| `ls -al` | a：当前目录下的所有文件，包括隐藏文件 <br/> l：显示详细信息，包括大小，属组，创建时间 |
