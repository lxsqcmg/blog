语法为 cal [month] [year]

# 文本翻页

翻页功能，在文本界面执行某些指令时，输出的信息很长，导致前面的部分已不再当前屏幕中了， 就可以使用该按键来前后翻页

# 上一页
shift + Page UP
# 下一页
shift + Page Down
